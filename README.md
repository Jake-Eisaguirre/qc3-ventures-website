# QC3E Ventures Website

A professional 3-page website for QC3E Ventures, a venture studio and holding company founded by three brothers — Lew, Jake, and Joseph Eisaguirre.

## Overview

This is a static HTML website designed to match the caliber of top-tier holding companies like Andreessen Horowitz, Sequoia Capital, and Founders Fund. The design emphasizes clean, confident, and credible presentation suitable for investor meetings and professional networking.

## Site Structure

```
qc3e-website/
├── index.html          # Homepage / Landing page
├── portfolio.html      # Portfolio companies and investments
├── team.html          # Team profiles and expertise
├── css/
│   └── main.css       # Complete styling system
├── js/
│   └── main.js        # Interactive enhancements
├── images/            # Image assets (placeholder directory)
└── README.md          # This file
```

## Pages

### 1. Homepage (`index.html`)
- Hero section with company name and tagline
- Mission statement and investment thesis
- Brief founder introduction
- Call-to-action for investor inquiries

### 2. Portfolio (`portfolio.html`)
- **TerraFirma Robotics**: Autonomous robotics (stealth mode)
- **Zaca Preserve**: Land conservation project (in development)
- Investment criteria and partnership opportunities

### 3. Team (`team.html`)
- Detailed founder profiles
- Combined expertise and experience
- Investment approach and contact information

## Brand Identity

- Navy: `#1E3A8A` (Primary)
- Charcoal: `#0F172A` (Text)
- Gold: `#CA8A04` (Accent)
- Warm White: `#F8FAFC` (Background)

**Typography:**
- Font: Inter (Google Fonts)
- Clean, professional hierarchy
- Swiss modernism inspired

**Design Principles:**
- Minimal, confident design
- Generous whitespace
- High contrast for accessibility
- Mobile-first responsive design

## Setup & Usage

### Local Development
1. Open `index.html` in a web browser
2. Navigate between pages using the header navigation
3. All assets are self-contained (no external dependencies except Google Fonts)

### Browser Compatibility
- Modern browsers (Chrome, Firefox, Safari, Edge)
- Mobile responsive design
- Accessible (WCAG AA compliant)
